# Secure Login System using CIA & AAA Model

## Project Description
This project is a Secure Login System developed using PHP and MySQL. It implements CIA (Confidentiality, Integrity, Availability) and AAA (Authentication, Authorization, Accounting) security models.

## Technologies Used
- PHP
- MySQL
- XAMPP Server

## Security Models

### CIA Model
- Confidentiality: Protects user data using secure password handling
- Integrity: Ensures data is not modified without authorization
- Availability: System is accessible when needed

### AAA Model
- Authentication: Verifies user identity during login
- Authorization: Controls user access to pages
- Accounting: Tracks user activity

## Features
- User Registration
- Secure Login System
- Session Management
- Access Control
